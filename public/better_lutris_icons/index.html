How to get better Lutris game icons in GNOME

Lutris, by default, makes your game icon blurry when you choose to add its icon to the GNOME "Show Applications" menu.

Unfortunately, it seems not even the Lutris dev team knows why they do it like that:

Nevertheless, it's not particularly difficult to fix this issue and have a nice-looking application menu again.

First, let Lutris make a desktop shortcut for your game. In Lutris, right-click the game of your choice and select Create desktop shortcut.

Lutris will create a .desktop file on your desktop. The next step is to adjust the shortcut to have the high-res icon you provided for your game. Open the file Lutris created with a text editor and change its icon entry to point to your icon file. For example, in my case I changed icon defining the line to:

Icon=/home/hbs18/Downloads/MW3.png

After setting the correct icon, you have to copy the shortcut to GNOME's shortcuts folder. To populate the list of apps in the applications menu, GNOME loads .shortcut files from /usr/share/applications. You need to move the shortcut file to this directory. Open a terminal in the directory where your shortcut file resides and use the moves command to move your shortcut there (with elevated permissions, of course). In my case, the command was:

hbs18@y700:~$ sudo mv net.lutris.mw3-3.desktop /usr/share/applications

The last step is to set appropriate permissions so the app can show in the applications menu. In my case, I used the following commands:

hbs18@y700:~$ cd /usr/share/applications
hbs18@y700:~$ sudo chmod 777 net.lutris.mw3-3.desktop
